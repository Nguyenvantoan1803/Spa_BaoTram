Đặt file logo tại đây với tên: logo.png

Cách làm:
1. Lưu ảnh logo Bảo Trâm (ảnh bạn đã gửi) vào thư mục này.
2. Đổi tên file thành: logo.png
   => Đường dẫn đầy đủ: d:\spa-baotram\frontend\public\logo.png
3. Lưu xong, trình duyệt sẽ tự cập nhật (Vite hot-reload). Nếu chưa thấy, nhấn Ctrl+F5.

Vite phục vụ thư mục public ở gốc, nên trong code logo được gọi là "/logo.png".
Có thể dùng .jpg/.webp cũng được, nhưng nhớ sửa lại tên trong Navbar.jsx cho khớp.
